# network-project
🌐 Network Lab Project – Packet Tracer
📌 Overview

This project is a network lab implemented using Cisco Packet Tracer.
It demonstrates basic networking concepts including device configuration, IP addressing, and connectivity between different network components.

🛠️ Technologies Used

Cisco Packet Tracer

Networking Fundamentals (Routing & Switching)

📂 Project Description

The lab includes:

Routers configuration

Switches connection

End devices (PCs / Servers)

IP addressing for all devices

Testing connectivity using ping

⚙️ Configuration Details

Assign IP addresses to all devices

Configure router interfaces

Enable communication between networks

Verify connectivity between devices

▶️ How to Run

Open the file using Cisco Packet Tracer

Load the .pkt file

Check configurations on routers and switches

Use ping from PCs to test connectivity

🎯 Objectives

Understand basic network topology

Practice IP configuration

Learn how devices communicate in a network

Apply routing concepts
